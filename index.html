Basics: HTML

Html (Hyper texting Markup Language) is a script where you can write general code of text, heading, any embeded code and much much more!
Html codes are mainly started by adding '<' and ended with '>'
Examples: 'h1' 'br' 'center' etc

Basics: CSS


CSS (Cascading Style Sheets) is a script where you can edit your code made in HTML (Hyper Text Markup Language) like adding colors, change size of text, upload a background image and much much more!

CSS mainly starts by adding '{' after writing the variable name and ends with '}' after writing the code for the variable

Example: 'background-image' 'font-size' 'font-family' etc.
